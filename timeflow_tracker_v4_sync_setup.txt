TimeFlow v4 云同步简要设置

1. 本地直接双击打开：可离线使用，但不能 Google 登录同步。
2. 要跨设备自动同步：
   - 在 Firebase 创建项目
   - 启用 Authentication -> Google
   - 启用 Firestore Database
   - 新建 Web App，复制配置 JSON
   - 把 HTML 上传到 GitHub Pages / Netlify / Vercel 等 HTTPS 地址
   - 在 Firebase Authorized domains 加上你的托管域名
   - 打开网页 -> 分类设置 -> 云同步（Firebase）
   - 粘贴配置 JSON -> 保存配置 -> 初始化云端 -> Google 登录
   - 勾选“启用自动同步”

3. 数据说明
   - 本地数据仍保存在浏览器 localStorage
   - 云端推送/拉取会同步整份状态
   - 建议首次使用前先导出一次 JSON 备份
