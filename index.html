本地模式
使用步骤
  1. 在 Firebase 创建项目，启用 Authentication 里的 Google 登录。
  2. 启用 Firestore Database。
  3. 把当前 HTML 上传到一个 HTTPS 网站，并把该域名加入 Firebase Authorized domains。
  4. 把 Web App 配置 JSON 粘贴到左侧，保存并初始化。
